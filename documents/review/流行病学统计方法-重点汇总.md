## 一、样本含量估计

### 1. 概念【名词解释】

样本含量估计指在研究设计阶段，根据研究目的和条件，确定所需的最少观察对象数目。

### 2. 影响因素（5个）【简答题】

| 影响因素 | 含义 | 典型取值 |
| --- | --- | --- |
| 检验水准 $\alpha$ | 第一类错误概率 | 双侧 $Z_\alpha = 1.96$，单侧 $Z_\alpha = 1.645$ |
| 检验效能 $1-\beta$ | 把握度 | $\beta = 0.10$ → $Z_\beta = 1.282$ |
| $\delta$（delta） | 两率或两均数之差 | — |
| $\sigma$（sigma） | 标准差 | 一般取较大值 |
| 单双侧 | 影响 $Z_\alpha$ 取值 | — |

用率计算时：暴露组$p_1$、对照组$p_0$，平均率$= \frac{p_1 + p_0}{2}$

### 3. 计算题【计算题】

各类研究均可能考样本含量计算，具体哪2种不确定，以下5种都有可能：

| 研究类型 | 公式涉及参数 |
| --- | --- |
| 横断面研究 | $p_1$、$p_0$、$\delta$、$\alpha$、$\beta$ |
| 病例对照研究 | $p_1$、$p_0$、$OR$、$\alpha$、$\beta$（不匹配 vs 匹配公式不同） |
| 队列研究 | $p_1$、$p_0$、$RR$、$\alpha$、$\beta$ |
| 筛检诊断试验 | 灵敏度、特异度、患病率等 |
| 实验研究 | $p_1$、$p_0$、$\delta$、$\alpha$、$\beta$ |

公式在最后一页提供，需要做到：知道什么研究用什么公式、公式里每个参数代表什么、带数进去算对。

## 二、描述性研究

### 1. 横断面研究【名词解释】

在某一时间点（或短时间段内），同时收集人群的暴露和疾病信息，描述其分布特征，不区分时间先后。

### 2. 4种抽样方法【简答题/选择题】

4种：单纯随机抽样、系统抽样、分层抽样、整群抽样。

| 方法 | 做法 | 精确度 | 成本/操作 | 适用场景 |
| --- | --- | --- | --- | --- |
| 单纯随机抽样 | 每个个体等概率被抽中 | 较高 | 样本大时操作麻烦 | 总体小、有完整名册 |
| 系统抽样 | 按固定间隔抽（如每隔10人抽1个） | 一般 | 简单省力 | 有顺序编号的名单 |
| 分层抽样 | 先按特征分层，再从各层随机抽 | 最高 | 较费力 | 层间差异大、层内差异小 |
| 整群抽样 | 随机抽取若干群，群内全部调查 | 最低 | 省时省钱 | 群间差异小、群内差异大 |

**选择题考点**：

- 精确度最高：分层抽样
- 最费时费力：分层抽样
- 误差最大：整群抽样

### 3. 常见偏倚

- 选择偏倚：研究对象的选取方式导致样本不能代表目标人群。
- 信息偏倚：收集信息时产生的系统误差，如回忆偏倚、报告偏倚。

## 三、病例对照研究

### 1. 名词解释【名词解释】

| 术语 | 定义 |
| --- | --- |
| 病例对照研究 | 选定病例组和对照组，回顾比较过去暴露情况，分析暴露与疾病的关联 |
| 现况研究 | 同横断面研究，某一时点调查人群中疾病和暴露的分布情况 |
| 描述性研究 | 描述疾病或健康状态在人群中的分布特征，不设对照组，不分析因果 |
| 暴露 | 研究对象接触过某物质或具备某特征/行为 |

### 2. 病例对照研究特点【简答题】

回顾性研究，由果及因，可同时研究多个暴露因素，适合罕见病研究。

### 3. OR值（比值比）【计算题/选择题】

**计算公式**：

- 不匹配：$OR = \frac{ad}{bc}$
- 匹配：$OR = \frac{b}{c}$（与a、d无关）

**表格格式陷阱**【选择题】：横纵标目互换后，a、b、c、d位置随之改变，直接套公式会算错。拿到题目先确认"行是啥、列是啥"，再按标准格式填数。

标准格式一（行=暴露，列=病例/对照）——最常见：

| | 病例 | 对照 |
| --- | --- | --- |
| 暴露(+) | a | b |
| 暴露(-) | c | d |

标准格式二（行=病例/对照，列=暴露）——横纵颠倒：

| | 暴露(+) | 暴露(-) |
| --- | --- | --- |
| 病例 | a | b |
| 对照 | c | d |

两种格式公式都是 $OR = \frac{ad}{bc}$，只是a、b、c、d的位置不同。

**文字逻辑法**（推荐）：不管表格怎么摆，按文字逻辑算，不记字母位置——

> 病例组暴露比值 ÷ 对照组暴露比值 = OR

步骤：①找病例组总人数和暴露/非暴露人数 → ②算病例组暴露比值 → ③找对照组 → ④算对照组暴露比值 → ⑤两者相除。

**举例（含糖饮料与龋齿）**：选取200名龋齿患儿（病例组）和200名健康儿童（对照组），回溯过去2年喝饮料习惯：

| 分组 | 每天喝含糖饮料 | 从不喝 | 合计 |
| --- | --- | --- | --- |
| 龋齿患儿（病例组） | a=150 | c=50 | 200 |
| 健康儿童（对照组） | b=50 | d=150 | 200 |

$OR = \frac{150 \times 150}{50 \times 50} = 9$

**文字逻辑验证**：病例组暴露比值 = 150 ÷ 50 = 3，对照组暴露比值 = 50 ÷ 150 ≈ 0.333，OR = 3 ÷ 0.333 = 9，与公式结果一致。

解读：龋齿患儿过去每天喝含糖饮料的比例是健康儿童的9倍，表明两者存在强关联。病例对照研究无法计算发病率，OR在疾病不罕见时会高估真实RR。

**表格行列互换后**（行=分组，列=暴露，意思完全一样）：

| 分组 | 每天喝含糖饮料 | 从不喝含糖饮料 |
| --- | --- | --- |
| 龋齿患儿（病例组） | 150 | 50 |
| 健康儿童（对照组） | 50 | 150 |

仍用文字逻辑：病例组暴露比值 = 150 ÷ 50 = 3，对照组暴露比值 = 50 ÷ 150 ≈ 0.333，OR = 3 ÷ 0.333 = 9。结果一样，只要认准"谁是病例组、谁是对照组"就不会算错。

**匹配病例对照研究的 OR**（特殊情况）：

匹配设计中每个病例配一个同年龄、同性别等的对照，结果用对子表呈现：

| 对照暴露情况 | 病例暴露情况 | 对子数 |
| --- | --- | --- |
| 暴露 | 暴露 | 30 |
| 暴露 | 未暴露 | 20 |
| 未暴露 | 暴露 | 60 |
| 未暴露 | 未暴露 | 90 |

匹配 OR 公式：$OR = \frac{b}{c}$，即"病例暴露而对照未暴露的对子数 ÷ 病例未暴露而对照暴露的对子数"。

计算：$OR = \frac{60}{20} = 3$。注意：30 和 90 这两个暴露情况一致的对子不参与计算。

### 4. 混杂因素判断【选择题】

**例题**：年龄与服用避孕药的关系（<40岁 vs ≥40岁）

- 计算年龄分层后的OR值
- 与原OR值比较：相等→无混杂；不相等→有混杂

### 5. 剂量反应关系【选择题】

**例题**：吸烟量（0、0.5、1、1.5、2包/天）与疾病关系

- 分别计算各剂量组的OR值
- 判断OR值是否随剂量增加而增大
- 若是→支持因果关联的证据之一

## 四、队列研究

### 1. 核心概念【名词解释】

- 队列研究：根据暴露状态将研究对象分为暴露组和非暴露组，随访观察两组疾病发生情况，比较发病率差异。
- 暴露：研究对象接触过某因素或具有某特征/行为。
- 队列：具有某种共同特征（如暴露于某因素）的一组人群。

### 2. 队列研究特点【简答题】

前瞻性研究，由因及果，可直接计算发病率和相对危险度。

### 3. 计算指标【名词解释/计算题】

| 指标 | 公式 | 含义 |
| --- | --- | --- |
| 相对危险度 RR | $RR = \frac{I_1}{I_0}$ | 暴露与疾病的关联强度 |
| 归因危险度 AR | $AR = I_1 - I_0$ | 暴露造成的超额发病风险 |
| 归因危险度百分比 AR% | $AR\% = \frac{I_1 - I_0}{I_1} \times 100\%$ | 暴露人群中归因于暴露的比例 |
| 人群归因危险度 PAR | $PAR = I_t - I_0$ | 暴露对全人群的影响 |
| 剂量反应关系 | RR 随剂量变化 | 因果关联的证据之一 |

**举例（含糖饮料与龋齿）**：招募2000名无龋齿小学生，分两组随访2年：

| 分组 | 患龋齿 | 未患龋齿 | 合计 | 发病率 |
| --- | --- | --- | --- | --- |
| 每天喝含糖饮料 | 150 | 850 | 1000 | 15% |
| 从不喝含糖饮料 | 50 | 950 | 1000 | 5% |

$RR = \frac{15\%}{5\%} = 3$

解读：每天喝含糖饮料的学生，2年内患龋齿的风险是不喝者的3倍。队列研究从暴露追踪结局，可直接计算发病率，RR直接反映风险倍数。

**表格行列互换后**（行=疾病，列=暴露，意思一样）：

| 患龋齿情况 | 每天喝饮料 | 从不喝饮料 | 合计 |
| --- | --- | --- | --- |
| 患龋齿 | 150 | 50 | 200 |
| 未患龋齿 | 850 | 950 | 1800 |

仍用文字逻辑：暴露组（每天喝饮料）总人数 = 150 + 850 = 1000，发病人数 = 150，发病率 = 150/1000 = 0.15；非暴露组（从不喝）总人数 = 50 + 950 = 1000，发病人数 = 50，发病率 = 50/1000 = 0.05；RR = 0.15 ÷ 0.05 = 3。结果一样。

### 4. 与实验研究的区别

| 特征 | 队列研究 | 实验研究 |
| --- | --- | --- |
| 处理因素 | 不施加，只观察自然暴露 | 人为施加干预措施 |
| 研究时序 | 前瞻性 | 前瞻性 |
| 分组方式 | 按已有暴露状态分组 | 随机分组 |
| 伦理限制 | 无干预，伦理限制少 | 需知情同意，伦理审查严格 |
| 因果论证强度 | 较强 | 最强 |

## 五、实验流行病学【名词解释】

### 1. 三种实验类型【判断题/选择题】

| 类型 | 研究对象 |
| --- | --- |
| 临床试验 | 个体 |
| 现场试验 | 个体 |
| 社区试验 | 人群 |

**判断题考点**：区分个体 vs 人群

### 2. 对照【选择题】

设立对照组是实验研究的基本原则，用于控制混杂因素，使实验组和对照组具有可比性。

### 3. 评价指标

| 指标 | 计算公式 |
| --- | --- |
| 有效率 | $\frac{\text{治疗有效人数}}{\text{治疗总人数}} \times 100\%$ |
| 治愈率 | $\frac{\text{治愈人数}}{\text{治疗总人数}} \times 100\%$ |
| 病死率 | $\frac{\text{因某病死亡人数}}{\text{患该病总人数}} \times 100\%$ |
| 不良事件发生率 | $\frac{\text{发生不良事件人数}}{\text{观察总人数}} \times 100\%$ |
| 生存率 | $\frac{\text{活过某时间点人数}}{\text{观察总人数}} \times 100\%$ |
| 保护率 | $\frac{\text{对照组发病率} - \text{实验组发病率}}{\text{对照组发病率}} \times 100\%$ |

### 4. 三原则【选择题】

| 原则 | 作用 |
| --- | --- |
| 对照 | 排除非研究因素干扰 |
| 随机 | 使各组基线可比 |
| 重复 | 保证结果可重复 |

**易混淆选项**：

- 均衡：是随机和对照的结果（各组基线可比），不是独立原则
- 盲法：是减少信息偏倚的手段，属于实验设计要求，不是三原则之一

## 六、筛检诊断试验

### 1. 筛检与诊断试验【名词解释】

- 筛检：用快速简便的试验方法，从表面健康的人群中找出可能患病的个体，属于初步筛查。
- 诊断试验：对筛检阳性者或可疑患者进一步检查以确诊或排除疾病。
- 区别：筛检面向健康人群、目的为发现可疑者、对灵敏度要求高；诊断试验面向可疑患者、目的为确诊、对特异度要求高。

### 2. 四格表（基础）

| - | 有病 | 无病 | 合计 |
| --- | --- | --- | --- |
| 检测阳性 | TP（真阳性） | FP（假阳性） | TP+FP |
| 检测阴性 | FN（假阴性） | TN（真阴性） | FN+TN |
| 合计 | TP+FN | FP+TN | N |

### 3. 真实性指标【选择题】

| 指标 | 公式 |
| --- | --- |
| 灵敏度（真阳性率） | $Se = \frac{TP}{TP+FN}$ |
| 特异度（真阴性率） | $Sp = \frac{TN}{FP+TN}$ |
| 假阳性率 | $FPR = \frac{FP}{FP+TN} = 1 - Sp$ |
| 假阴性率 | $FNR = \frac{FN}{TP+FN} = 1 - Se$ |
| 正确指数（约登指数） | $Se + Sp - 1$ |
| 阳性似然比 | $\frac{Se}{1-Sp} = \frac{TP/(TP+FN)}{FP/(FP+TN)}$ |
| 阴性似然比 | $\frac{1-Se}{Sp} = \frac{FN/(TP+FN)}{TN/(FP+TN)}$ |

### 4. 可靠性指标

- 符合率：$P_o = \frac{TP+TN}{N}$
- 卡帕值（Kappa）：排除机遇一致后的一致性，$\kappa = \frac{P_o - P_e}{1 - P_e}$，$>0.75$为一致性好。

### 5. 预测值【选择题/判断题】

| 预测值 | 含义 | 公式 |
| --- | --- | --- |
| 阳性预测值 $PV_+$ | 阳性中真正患病的比例 | $\frac{TP}{TP+FP}$ |
| 阴性预测值 $PV_-$ | 阴性中真正未患病的比例 | $\frac{TN}{FN+TN}$ |

用贝叶斯展开：$PV_+ = \frac{Se \times P}{Se \times P + (1-Sp) \times (1-P)}$

**重要结论**【选择题/判断题】：

- 灵敏度、特异度一定时：患病率$\downarrow$ → 阳性预测值$\downarrow$，阴性预测值$\uparrow$
- 患病率不变时：灵敏度$\downarrow$ → 特异度$\uparrow$

### 6. ROC曲线【判断题】

- 横轴：1-特异度
- 纵轴：灵敏度

### 7. 串联与并联【选择题/判断题】

| 方式 | 判定规则 | 灵敏度 | 特异度 |
| --- | --- | --- | --- |
| 串联（系列试验） | 全部阳性才判阳性 | 降低 | 提高 |
| 并联（平行试验） | 任一阳性即判阳性 | 提高 | 降低 |

### 8. 常见偏倚

| 偏倚类型 | 含义 |
| --- | --- |
| 沾染偏倚 | 对照组接受了实验组干预，差异缩小 |
| 干扰偏倚 | 实验组额外接受其他措施，夸大效果 |
| 接受干预偏倚 | 未按分组接受干预，实际分析与设计不符 |
| 随访时间偏倚 | 各组随访时间不一致，影响结局判断 |
| 入院率偏倚（伯克森偏倚） | 住院患者不代表一般人群，导致虚假关联 |
| 志愿者偏倚 | 志愿者与非志愿者特征不同，影响结果外推 |

## 七、临床预后研究

### 1. 概念

研究疾病确诊后的结局（痊愈、复发、致残、死亡等）及其影响因素。

### 2. 常用指标

| 指标 | 含义 |
| --- | --- |
| 生存率 | 活过某时间点的比例 |
| 中位生存时间 | 50%患者死亡的时间点 |
| 复发率 | 病情缓解后再次发作的比例 |

### 3. 分析方法

生存分析为主：寿命表法、Kaplan-Meier法。

## 八、病因推断

### 1. 建立病因假设（3种方法）

| 方法 | 核心要点 |
| --- | --- |
| 假设演绎法 | 从一般原理推导具体假设 |
| Mill准则 | 求同、求异、共变、类推、排除（5种） |
| 类比推理法 | 相似疾病可能有相似病因 |

### 2. Hill因果推断准则（9条）【名词解释】

- **必须满足**：时序性（原因在结果之前）
- **重要**：关联强度、剂量反应关系、一致性
- **支持性**：生物学合理性、生物学一致性、特异性、实验证据、相似性

注意：特异性在现代多因素疾病中较少强调。

## 九、循证医学【名词解释】

### 1. 定义

最佳证据 + 临床经验 + 患者价值，三者结合制定决策。

### 2. 实践5步

提出问题 → 检索证据 → 评价证据 → 应用证据 → 后效评价

### 3. PICO格式

## 十、系统综述与Meta分析

### 1. 概念 【名词解释】

| 术语 | 定义 |
| --- | --- |
| 系统综述 | 系统收集、评价、合成研究，得出综合结论 |
| Meta分析 | 多个研究结果的定量合成方法 |

### 2. 常见偏倚

| 偏倚 | 含义 |
| --- | --- |
| 发表偏倚 | 阳性结果更易发表，高估疗效 |
| 选择偏倚 | 纳入研究的选择不全面 |
| 语言偏倚 | 仅纳入某语种文献造成的偏倚 |

### 3. 敏感性分析

检验结论稳健性，剔除某些研究后看结论是否改变。
